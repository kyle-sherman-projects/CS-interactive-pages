📖 Key Vocabulary / Vocabulario Clave

input() — Asks the user to type something / Pide al usuario que escriba algo

float() — Converts text → decimal number / Convierte texto → número decimal

int() — Converts text → whole number / Convierte texto → número entero

str() — Converts number → text / Convierte número → texto

+ (strings) — Joins text together / Une textos

* (numbers) — Multiplies numbers / Multiplica números

📝 Program Purpose
This program calculates total ingredient amounts based on servings.
Este programa calcula la cantidad total de ingredientes según las porciones.
"""
This program asks the user for three ingredients,
three amounts, and a number of servings, and
determines how much of each ingredient is needed
to serve the specified number of servings.
"""
🔁 REPEAT FOR EACH INGREDIENT (×3) / REPETIR PARA CADA INGREDIENTE
User Input
Step 1 — Get Ingredient Name
Ask the user to type an ingredient name. Python saves it as a string (text).
Pedir al usuario el nombre del ingrediente. Python lo guarda como texto (string).
ingredient_1 = input("Enter ingredient 1: ")
What happens

User types: flour → ingredient_1 now holds "flour" (a string)

Input Type Conversion String +
Step 2 — Get Ounces (3 Concepts in 1 Line!)
This line does three things at once. Let's break it apart.
Esta línea hace tres cosas a la vez. Vamos a separarla.
ingredient_1_ounces = float(input("Enter ounces for " + ingredient_1 + ". "))
🔍 Breaking it down — inside → outside

1. "Enter ounces for " + ingredient_1 + ". "
   String operator + joins text → "Enter ounces for flour. "

2. input(...) shows that message and waits → user types 2.5 → returns "2.5" (still a string!)

3. float(...) converts "2.5" → 2.5 (now a decimal number we can use in math)

↕ Steps 1–2 repeat for ingredients 2 and 3
Input Type Conversion
Step 3 — Get Number of Servings
Ask how many servings. Convert to integer (whole number) since you can't have 2.5 servings in this recipe.
Preguntar cuántas porciones. Convertir a entero (número entero) porque no puedes tener 2.5 porciones.
num_servings = int(input("Number of servings: "))
What happens

User types: 4 → input() returns "4" (string) → int() converts to 4 (integer)

Why int() not float()? Servings are whole numbers. int() = whole, float() = decimal.

Math * str() String + Output
Step 4 — Calculate & Display Results
The most complex line: multiply, convert the answer to text, join it all together, and print.
La línea más compleja: multiplicar, convertir la respuesta a texto, unir todo e imprimir.
print("Total ounces of " + ingredient_1 + ": " + str(ingredient_1_ounces * num_servings))
🔍 Breaking it down — inside → outside

1. ingredient_1_ounces * num_servings
   Math: 2.5 * 4 = 10.0 (a float number)

2. str(10.0)
   Convert number → string: "10.0" (now it's text — we need this to use + with other strings!)

3. "Total ounces of " + "flour" + ": " + "10.0"
   String + joins all text → "Total ounces of flour: 10.0"

4. print(...) displays the result on screen.

⚠️ Why do we need str() here?

Python cannot use + to join a string and a number. You'll get a TypeError!
We must convert the number to text first with str().
Python no puede usar + para unir texto con número. ¡Primero convierte con str()!

🧠 Key Concepts Summary / Resumen de Conceptos Clave

User Input

input() always returns a string, even if the user types a number.
input() siempre devuelve texto.

Type Conversion

float() → decimal number
int() → whole number
str() → text
Cambiar el tipo de dato.

String Operator +

Joins text together. Both sides must be strings.
"hi" + "there" → "hithere"
Une textos. Ambos lados deben ser strings.

Common Error

TypeError when you mix string + number. Fix: wrap the number in str().
Error común: mezclar texto + número. Solución: usar str().
