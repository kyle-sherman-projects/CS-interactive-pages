Section 4: Mini-Project — Number Guessing Game (25 min)

Sección 4: Mini-Proyecto — Juego de Adivinanza
Specification

1 Set a secret number (e.g., 7)

2 Max 5 attempts

3 After wrong guess: higher/lower hint

4 Correct → "You win!" + attempt count

5 5 failures → "Game over!" + reveal number

🧠 PLAN first
Run it / Córrelo — Code Block L3G
Trinket setup: Empty — student builds number guessing game
secret = 7
attempts = 0
won = False
while attempts < 5 and not won:
guess = int(input("Guess (1-10): "))
attempts = attempts + 1
if guess == secret:
print("You win! Took " + str(attempts) + " attempts.")
won = True
elif guess < secret:
print("Higher!")
else:
print("Lower!")
if not won:
print("Game over! It was " + str(secret))

This counter + early exit pattern is exactly what you'll use in Password Authenticator.
