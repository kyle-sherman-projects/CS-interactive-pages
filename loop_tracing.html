Section 3: Spot the Bug (12 min)

Sección 3: Encuentra el Error

Find the bug by reading only. Then verify with Trinket.

🐛 Bug 1 Infinite Loop
📋 Buggy Code 1
1x = 10
2while x > 0:
3 print(x)
4 x = x + 1
🔮 What's the bug? How do you fix it?

Bug: Line 4 adds instead of subtracts. x grows forever. Fix: x = x - 1
