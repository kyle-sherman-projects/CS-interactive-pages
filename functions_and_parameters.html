Recipe & Ingredients — A recipe is the function. Ingredients like flour, sugar, and eggs are the parameters. Each time you bake, you plug in specific amounts (arguments).

Review: Single Parameters

Before we go further — let's confirm what we already know.

Function Anatomy — Review
def double_number(x):        # x is the PARAMETER
    double_x = 2 * x
    print(double_x)

double_number(10)    # 10 is the ARGUMENT → x gets the value 10
Term / TérminoWhat it isIn the code above
def function_name(parameter): Defines a function; the variable inside () is the parameter def double_number(x):
function_name(argument) Calls the function; the value inside () is the argument double_number(10)
Parameter variable Gets its value from the argument at call time x = 10 (inside the function)
The Problem: What if we need more than one input?

Think back to our recipe analogy. A recipe for cookies doesn't just need flour — it also needs sugar, eggs, and more.

A function that only accepts one parameter is limited. In today's lesson, we'll learn how Python lets us pass multiple inputs into a single function.

En español: Hoy aprenderemos cómo una función puede recibir múltiples parámetros — no solo uno.

Step-by-Step: How Python Reads a Multi-Parameter Call
def print_sum(first, second):
    print(first + second)

print_sum(5, 6)
1
def print_sum(first, second):
Python reads the definition — it knows this function needs TWO inputs.
Python lee la definición — sabe que esta función necesita DOS entradas.
2
print_sum(5, 6)
Python calls the function. It matches arguments to parameters in order: first=5, second=6.
Python llama la función. Empareja argumentos con parámetros en orden: first=5, second=6.
3
print(first + second) → print(5 + 6) → 11
Inside the function, first=5 and second=6, so 5+6 = 11 is printed.
Dentro de la función, first=5 y second=6, entonces 5+6 = 11 se imprime.
